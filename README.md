# Quickstart

Please enter the commands below in your system terminal/command line to run the app.

## First Time

```
git clone https://github.com/wmeikle33/Float-Visualization-App.git
cd Float-Visualization-App
python -m venv .venv
source .venv/bin/activate # Windows: .\.venv\Scripts\Activate.ps1
pip install -e .
python scripts/run_app.py
```

## Subsequent Runs

```
cd Float-Visualization-App
python -m venv .venv
source .venv/bin/activate # Windows: .\.venv\Scripts\Activate.ps1
python scripts/run_app.py

```

## Reset Data

```
cd Float-Visualization-App
python -m venv .venv
source .venv/bin/activate # Windows: .\.venv\Scripts\Activate.ps1
python scripts/reset_app.py

```
